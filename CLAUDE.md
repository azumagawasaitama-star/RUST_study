**第1章で出たQ&A：**
- タプルとは → 異なる型をまとめるもの。配列は同型固定長、タプルは複数型固定長
- `usize` / `isize` → ポインタサイズ整数。配列インデックスは必ず `usize`
- インライン展開 → 定数はコンパイル時に値に直接置き換えられる（メモリ節約）
- スネークケース → `my_variable` 形式。コンパイラが警告を出す公式推奨慣習
- デストラクチャリング → `let (a, b) = タプル` で変数に直接分解して代入
- `{:?}` → デバッグ用表示。タプルや配列など `{}` で表示できないものに使う
- 警告（warning） → エラーではない。未使用変数には `_x` と先頭に `_` をつけると消える
- 配列の長さはコンパイル時確定 → 可変長にしたい場合は後の章の `Vec` を使う
